# Liquid Glass UI Components

A modern **Liquid Glass UI kit** featuring smooth gradients, blur effects, and premium interface components.

Inspired by glassmorphism and modern design trends.

## ✨ Features

- Liquid glass style UI
- Soft gradient backgrounds
- Blur / glassmorphism effects
- Modern rounded components
- Responsive layout
- Clean and minimal design

## 📦 Components

This UI kit includes:

- Buttons
- Icon Buttons
- Text Fields
- Select Dropdown
- Toggle Switch
- Tabs
- Cards
- UI Panels

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- CSS backdrop-filter
- Modern UI layout

## 📱 Responsive

The components are designed to work on:

- Desktop
- Tablet
- Mobile
